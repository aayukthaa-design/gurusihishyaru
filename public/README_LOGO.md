# Logo Setup Instructions

## Place the Guru Shishyaru Tutorials logo here

1. Save the uploaded Guru Shishyaru Tutorials logo image as:
   
   `public/logo.png`

2. The logo will automatically appear in:
   - Login page (left panel + mobile header)
   - Sidebar header
   - Dashboard welcome banner (watermark)
   - Unauthorized page
   - Browser tab favicon
   - Apple touch icon

3. Supported formats: PNG (recommended), JPG, WEBP, SVG

4. If you want to use a different filename or format, update the `src="/logo.png"` 
   path in: `src/app/components/BrandLogo.tsx`
